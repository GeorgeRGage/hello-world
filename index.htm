HR Premier® is the most Complete and Aaffordable payroll, human resource and back office accounting software solution for companies of every size. It is a true 100% web-based system, which allows users to access, process, track, update all information from anywhere, anytime through their web browser, both Internet or intranet. HR PremierT provides audit trails to view all changes to check data accuracy and has powerful reporting capabilities that are easy to use with over 100 standard reports.

HR Premier® Payroll allows designated users to process payrolls, which can be accomplished from the local office or remote sites for all employees including those in multi-state jurisdictions. Payroll tracks workers compensation, 401k distributions as well as federal, state and local new hires. Payroll provides standard and customized reports, full tax tables with quarterly tax updates, unlimited direct deposits, unlimited deductions, labor distribution, project management costing, back office invoicing and point of sale interface for direct capture of time and attendance and designed to meet complex payroll requirements for Corporations, payroll service companies, professional employer organizations (PEO's), temporary staffing organizations, CPA firms and Administrative Services Organizations (ASO).

 

 

HR Premier® HR includes all data entry and corresponding reports for personnel tracking, applicant tracking, labor relations tracking, benefits administration, compensation administration, education/skills, full ISO compliance tracking, health and safety and time and attendance tracking. HR provides for employee self service and supervisor self service. It maintains EEO, AA, COBRA, OSHA, ADA, I-9 and FMLA reporting requirements.

HR Premier® Back Office Accounting is designed for the PEO, ASO and Service Bureau's which require tracking, reporting and invoicing, enterprise level payables (such as garnishment payments, one check multiple stubs listing all employees) and general accounting.

Technology: HR Premier® was designed, developed, and optimized exclusively for the Microsoft platform, allowing a core of service and support expertise particular to this environment. This includes Internet Explorer, Internet Information Server, and SQL Server. HR PremierT is a true three-tier client-server application. The three tiers break down to thin client, middle object broker, and back end data store. These are: thin client->Internet Explorer (IE), object request broker > Internet Information Server (IIS), and backend data store > SQL Server (SQL).

When a user accesses HR Premier®, the thin client submits a request to the object request broker. This object request broker accepts the request and after verifying the users security access, passes the request to the backend data store.

The backend data store processes the request and returns the requested information to the object request broker. The object request broker then formats these results and returns them to the thin client as pure HTML. This type of design has numerous advantages. One is the ability to control all upgrades in a single place (the Internet Information Server). Another advantage is the standardization of user interfaces (the Web browser is the interface). A third advantage is the inherent security built into this type of design (users are verified on several layers). Further advantages include: performance, software licensing, connection pooling, remote access, built in e-mail/inet mail functionality, etc.
